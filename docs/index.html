DISCORD WHAT?

Discord is an all-in-one voice and text chat originally created for gamers,but now used by all since it's free, secure, and works on any desktop (using their dedicated client), or web browser, as well as on your smartphone, meaning the fun can continue even while you are on-the-go, or off on vacation.

Imagine all the advantages and benefits of WhatsApp™, TeamSpeak™, Skype™, plus the control of the good 'ol message boards.

Well, now you can stop imagining and start enjoying them in a single app!
